# Тесты для SlotMathModule

Этот проект содержит комплексные тесты для модуля математики слотовых игр.

## Структура тестов

### SuperballMathTests.cs
Тесты для математики игры SuperBall:
- Инициализация и конструктор
- Проверка RTP (Return to Player) для разных процентов
- Проверка лимитов выигрышей и овердрафта
- Проверка баланса и расчетов
- Определение джекпотов
- Распределение выигрышей
- Работа с разными номиналами

### SlotTests.cs
Тесты для базового класса Slot:
- Инициализация и загрузка игр
- Работа с кредитами (пополнение, снятие)
- Выполнение спинов
- Расчет ставок и линий
- Статистика и RTP
- Скрейч-режим
- Покупка билетов

### FeedTests.cs
Тесты для генерации ленты выигрышей:
- Генерация ленты
- Соблюдение процента возврата
- Распределение выигрышей
- Уникальность индексов
- Работа с разными процентами

### IntegrationTests.cs
Интеграционные тесты:
- Полные игровые сессии
- Стабильность RTP на длинных сессиях
- Работа нескольких игр независимо
- Бонусные игры
- Отслеживание баланса

## Запуск тестов

```bash
dotnet test
```

Или через Visual Studio: Test → Run All Tests

## Важные моменты

1. **RTP тесты** - используют большое количество спинов (10000+) для статистической точности
2. **Допуски** - RTP проверяется с допуском ±2-5% из-за случайности
3. **Длительность** - некоторые тесты могут выполняться несколько секунд из-за большого количества итераций
4. **Зависимости** - тесты требуют правильно настроенного проекта SlotMathModule
